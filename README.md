尝试照着一些设计稿做出相应的页面，不止是CSS的练习，但是更多专注于CSS。
练习时使用 WebStorm / Pug / CSS / Chrome73。

## 设计稿来源

- 02-Twitter，简单的Twitter个人页面复刻，因为刚学习CSS，有些地方也十分粗暴。使用了简单的圣杯布局，以及来自[ICON 8](https://icons8.com/)的图标。
